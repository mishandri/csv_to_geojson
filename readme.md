### Что делает скрипт
1. Читает CSV с разделителем ; (точка с запятой)
2. Извлекает название региона, тип, ID и координаты
3. Парсит строку с координатами как JSON
4. Меняет местами координаты из формата [latitude, longitude] в формат GeoJSON [longitude, latitude]
5. Создаёт структуру GeoJSON с правильной геометрией (MultiPolygon)
6. Сохраняет результат в новый файл

### Инструкция по использованию
1. Сохраните скрипт в файл, например csv_to_geojson.py
2. Убедитесь, что ваш CSV-файл находится в той же папке, или укажите полный путь к нему
3. Запустите скрипт из командной строки:
```sh
python csv_to_geojson.py имя_файла.csv
```

### Дополнительные возможности
- Указать выходной файл:
```sh
python csv_to_geojson.py имя_файла.csv имя_файла.geojson
```
- Обработка ошибок: Скрипт сообщит о проблемных строках и продолжит работу

- Прогресс: Показывает, какие регионы обработаны

- Статистика: В конце выводит количество обработанных регионов

### Примечание
В CSV от Яндекса координаты хранятся в формате [latitude, longitude], а GeoJSON требует [longitude, latitude]. Скрипт автоматически меняет их местами, чтобы карта отображалась правильно.
